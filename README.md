## 📖 Sobre o projeto

**Rental Manager** é um sistema desenvolvido para **gerenciar imóveis locados** de forma eficiente.  
O sistema tem como objetivo ajudar administradores a controlar contratos, finanças e gerar alertas para prazos importantes.

Principais funcionalidades planejadas:  
- **Gestão de contratos de locação:** acompanhar datas de início e término, valores de aluguel.  
- **Relatórios:** alertas para contratos prestes a vencer, atrasos de pagamento e lista de imóveis.  
- **Controle financeiro:** monitorar pagamentos, saldos e alugueis em atraso.  

O backend está atualmente em desenvolvimento e, futuramente, serão implementadas **funcionalidades avançadas de relatórios e análises**.

## 🛠 Tecnologias

- **Backend:** ASP.NET Core (.NET 9)  
- **Banco de dados:** PostgreSQL (hosted no Railway)  
- **Frontend (futuro):** HTML, CSS e JS
